Programming Begins Here

Computer programming seems very complex to most people and they have a point in thinking that way. It is. Computer programming is the language of a computer; in similar ways how we would learn another language to be able to communicate to others with different dialects. Now please be reminded that learning any new skill will take time, dedication, and resilience. While coding is something that can be learned it doesn’t necessarily make it easy.

Being able to deal with frustration when coding is an essential trait for a programmer. Major key ladies and gentlemen, time builds quality programmers. You will spend countless hours being frustrated with your laptop but trust me, it gets better as you calmly move through the calamities of code. It took me weeks to begin to actually understand the syntax. Syntax is the actual code that you would type in something called a “text editor” which is used to produce code. Think of the text editor as Microsoft Word and syntax being the words in the body of the paragraphs.
